##  О проекте

**ErrorChec** — это профессиональный инструмент для анализа синтаксических ошибок в исходном коде. Разработан для разработчиков, которые хотят быстро находить и исправлять ошибки без запуска полноценного компилятора. Программа поддерживает **10+ языков программирования**, обладает интуитивно понятным интерфейсом и множеством настроек для комфортной работы.

###  Основные возможности

####  Анализ кода
- **Автоматическое определение языка** по расширению файла (с подтверждением пользователя)
- **Глубокая проверка синтаксиса** для Python, PHP, JavaScript, Java, C++, Ruby, Go, Rust, Swift
- **Поддержка компиляторов** для точного анализа (PHP, Node.js, Java JDK, GCC, .NET Core)
- **Поиск типичных ошибок**: пропущенные точки с запятой, незакрытые скобки, ошибки отступов

####  Интерфейс и настройки
- **40+ тем оформления**: 20 темных и 20 светлых с разными оттенками
- **6 тем подсветки синтаксиса**: Default, Monokai, Solarized, Dracula, Nord, GitHub
- **6 кодировок**: UTF-8, UTF-16, CP1251, ISO-8859-1, ASCII, KOI8-R
- **Настраиваемое резервное копирование** с выбором интервала (от 10 до 3600 секунд)
- **Мониторинг ресурсов**: отображение использования памяти и CPU

####  Управление компиляторами
- Проверка наличия компиляторов при запуске
- Уведомление об отсутствии компилятора для языка
- Встроенный менеджер компиляторов с проверкой статуса
- Инструкции по установке для каждого языка

####  Работа с файлами
- Открытие и сохранение файлов с автоматическим определением расширения
- Сохранение как с выбором языка и расширения
- Автоматическая установка расширения в зависимости от языка
